# 如何调参:
观察`Generation_demo.py`中的参数设置，其中的`alpha`和`reimu`，可以调整生成图像的风格和细节, 分别调节三个参考图片在最终风格中的各自占比(三个变量两个自由度).

代码里有三个参数需要调节
- CFG: ω_1 控制text prompt的引导强度
- swap_guidance_scale: ω_2 控制style image的引导强度
- interpolation: Linear smoothing λ. 这个不一定要调参, 可以不动

主要修改前两个参数, 然后
```bash
python -u Generation_demo.py --cfg 7.5 --swap_guidance_scale 7.5
```
看输出的图片质量. 做一点小小的调节, 直到满意为止. 配好环境跑一遍你就知道输出在哪了...
